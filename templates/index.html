PKPY

Turn your python packages into native DEB or RPM packages.

Works with most packages on pypi, github and bitbucket. Try stream from pypi.

Go

Other examples

  • luigi (deb)
  • luigi (rpm)
  • pandas (deb)
  • pandas (rpm)

Build from Github

  • miku/gluish (deb)

Mode of operation

Enter the name of a Python repository into the box and hit enter. PKPY will create a download page for RPM, DEB and i386 or x86_64 when necessary. Download a single zipfile with all necessary packages and install them on you deployment server with standard tools like yum or dpkg.
